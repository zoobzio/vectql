# Quickstart

Get started with VECTQL in 5 minutes.

## Installation

```bash
go get github.com/zoobzio/vectql
go get github.com/zoobzio/vdml
```

## Define Your Schema

Create a VDML schema describing your vector collections:

```go
import "github.com/zoobzio/vdml"

schema := vdml.NewSchema("myapp").
    AddCollection(
        vdml.NewCollection("documents").
            AddEmbedding(
                vdml.NewEmbedding("content_embedding", 1536).
                    WithMetric(vdml.Cosine),
            ).
            AddMetadata(vdml.NewMetadataField("title", vdml.TypeString)).
            AddMetadata(vdml.NewMetadataField("source", vdml.TypeString)).
            AddMetadata(vdml.NewMetadataField("page", vdml.TypeInt)),
    )
```

## Create an Instance

Bind the schema to a VECTQL instance:

```go
import "github.com/zoobzio/vectql"

v, err := vectql.NewFromVDML(schema)
if err != nil {
    log.Fatal(err)
}
```

## Build Queries

### Semantic Search

```go
import "github.com/zoobzio/vectql/pkg/pinecone"

result, err := vectql.Search(v.C("documents")).
    Vector(vectql.Vec(v.P("query_embedding"))).
    Embedding(v.E("documents", "content_embedding")).
    TopK(10).
    Render(pinecone.New())
```

### Search with Filters

```go
result, err := vectql.Search(v.C("documents")).
    Vector(vectql.Vec(v.P("query_embedding"))).
    Embedding(v.E("documents", "content_embedding")).
    TopK(10).
    Filter(v.Eq(v.M("documents", "source"), v.P("source"))).
    Render(pinecone.New())
```

### Upsert Vectors

```go
record := vectql.NewRecord(v.P("id"), vectql.Vec(v.P("embedding"))).
    WithMetadata(v.M("documents", "title"), v.P("title")).
    WithMetadata(v.M("documents", "source"), v.P("source"))

result, err := vectql.Upsert(v.C("documents")).
    AddVector(record.Build()).
    Render(pinecone.New())
```

### Delete Vectors

```go
// By IDs
result, err := vectql.Delete(v.C("documents")).
    IDs(v.P("ids")).
    Render(pinecone.New())

// By filter
result, err := vectql.Delete(v.C("documents")).
    Filter(v.Eq(v.M("documents", "source"), v.P("source"))).
    Render(pinecone.New())
```

### Fetch Vectors

```go
result, err := vectql.Fetch(v.C("documents")).
    IDs(v.P("ids")).
    Render(pinecone.New())
```

## Using the Result

```go
result, err := query.Render(pinecone.New())
if err != nil {
    return err
}

// The JSON query for the provider
fmt.Println(result.JSON)

// Parameters that must be provided at execution time
fmt.Println(result.RequiredParams) // ["query_embedding", "source"]
```

## Choosing a Provider

```go
import (
    "github.com/zoobzio/vectql/pkg/pinecone"
    "github.com/zoobzio/vectql/pkg/qdrant"
    "github.com/zoobzio/vectql/pkg/milvus"
    "github.com/zoobzio/vectql/pkg/weaviate"
)

// Each provider renders to its native format
pineconeResult, _ := query.Render(pinecone.New())
qdrantResult, _ := query.Render(qdrant.New())
milvusResult, _ := query.Render(milvus.New())
weaviateResult, _ := query.Render(weaviate.New())
```
