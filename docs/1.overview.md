# VECTQL

VECTQL is a type-safe query builder for vector databases, designed for building semantic search, RAG, and AI applications.

## Features

- **Type-safe queries** — Compile-time validation against VDML schemas
- **Multi-provider support** — Pinecone, Qdrant, Milvus, Weaviate
- **Parameterized queries** — Safe, reusable query templates
- **Fluent API** — Intuitive builder pattern for constructing queries
- **Security by design** — Internal types prevent injection attacks

## Architecture

```
┌─────────────────────────────────────────────────────────────────┐
│                        Public API Layer                          │
│  vectql.Search(), vectql.Upsert(), instance.C(), instance.E()   │
└─────────────────────────────────────────────────────────────────┘
                               │
                               ▼
┌─────────────────────────────────────────────────────────────────┐
│                        Builder Layer                             │
│  Builder struct, method chaining, AST construction               │
└─────────────────────────────────────────────────────────────────┘
                               │
                               ▼
┌─────────────────────────────────────────────────────────────────┐
│                     Internal Types Layer                         │
│  VectorAST, Collection, EmbeddingField, MetadataField, Param    │
└─────────────────────────────────────────────────────────────────┘
                               │
                               ▼
┌─────────────────────────────────────────────────────────────────┐
│                       Renderer Layer                             │
│  Pinecone, Qdrant, Milvus, Weaviate providers                   │
└─────────────────────────────────────────────────────────────────┘
```

## Quick Example

```go
import (
    "github.com/zoobzio/vdml"
    "github.com/zoobzio/vectql"
    "github.com/zoobzio/vectql/pkg/pinecone"
)

// Define schema with VDML
schema := vdml.NewSchema("ecommerce").
    AddCollection(
        vdml.NewCollection("products").
            AddEmbedding(vdml.NewEmbedding("embedding", 1536).WithMetric(vdml.Cosine)).
            AddMetadata(vdml.NewMetadataField("category", vdml.TypeString)).
            AddMetadata(vdml.NewMetadataField("price", vdml.TypeFloat)),
    )

// Create validated instance
v, _ := vectql.NewFromVDML(schema)

// Build semantic search query
result, _ := vectql.Search(v.C("products")).
    Vector(vectql.Vec(v.P("query_vec"))).
    Embedding(v.E("products", "embedding")).
    TopK(10).
    Filter(v.Eq(v.M("products", "category"), v.P("category"))).
    Render(pinecone.New())

// result.JSON contains provider-specific query
// result.RequiredParams = ["query_vec", "category"]
```
