# Operators Reference

Complete reference for all VECTQL operators.

## Filter Operators

### Comparison Operators

| Operator | Method | Description | Pinecone | Qdrant | Milvus | Weaviate |
|----------|--------|-------------|----------|--------|--------|----------|
| EQ | `Eq()` | Equal to | `$eq` | `match` | `==` | `Equal` |
| NE | `Ne()` | Not equal to | `$ne` | `match` (negated) | `!=` | `NotEqual` |
| GT | `Gt()` | Greater than | `$gt` | `range.gt` | `>` | `GreaterThan` |
| GTE | `Gte()` | Greater or equal | `$gte` | `range.gte` | `>=` | `GreaterThanEqual` |
| LT | `Lt()` | Less than | `$lt` | `range.lt` | `<` | `LessThan` |
| LTE | `Lte()` | Less or equal | `$lte` | `range.lte` | `<=` | `LessThanEqual` |

**Example:**

```go
filter := v.Gte(v.M("products", "price"), v.P("min_price"))
```

### Set Membership Operators

| Operator | Method | Description | Pinecone | Qdrant | Milvus | Weaviate |
|----------|--------|-------------|----------|--------|--------|----------|
| IN | `In()` | Value in set | `$in` | `match.any` | `in` | `ContainsAny` |
| NIN | `NotIn()` | Value not in set | `$nin` | `must_not` | `not in` | `ContainsAny` (negated) |

**Example:**

```go
filter := v.In(v.M("products", "category"), v.P("categories"))
```

### Existence Operators

| Operator | Function | Description | Pinecone | Qdrant | Milvus | Weaviate |
|----------|----------|-------------|----------|--------|--------|----------|
| EXISTS | `Exists()` | Field exists | `$exists: true` | `is_empty: false` | - | `IsNull: false` |
| NOT_EXISTS | `NotExists()` | Field doesn't exist | `$exists: false` | `is_empty: true` | - | `IsNull: true` |

**Example:**

```go
filter := vectql.Exists(v.M("products", "discount"))
```

### String Operators

| Operator | Function | Description | Pinecone | Qdrant | Milvus | Weaviate |
|----------|----------|-------------|----------|--------|--------|----------|
| CONTAINS | `Contains()` | Contains substring | Limited | `match.text` | `like` | `Like` |
| STARTS_WITH | `StartsWith()` | Starts with prefix | - | `match.text` | `like` | `Like` |
| ENDS_WITH | `EndsWith()` | Ends with suffix | - | `match.text` | `like` | `Like` |
| MATCHES | `Matches()` | Regex match | - | - | - | - |

**Example:**

```go
filter := vectql.StartsWith(v.M("products", "sku"), v.P("prefix"))
```

### Array Operators

| Operator | Function | Description | Pinecone | Qdrant | Milvus | Weaviate |
|----------|----------|-------------|----------|--------|--------|----------|
| ARRAY_CONTAINS | `ArrayContains()` | Array contains value | `$in` | `match.any` | `array_contains` | `ContainsAny` |
| ARRAY_CONTAINS_ANY | `ArrayContainsAny()` | Contains any value | `$in` | `match.any` | `array_contains_any` | `ContainsAny` |
| ARRAY_CONTAINS_ALL | `ArrayContainsAll()` | Contains all values | - | `match.all` | `array_contains_all` | `ContainsAll` |

**Example:**

```go
filter := vectql.ArrayContains(v.M("products", "tags"), v.P("tag"))
```

## Logical Operators

| Operator | Method | Description | Pinecone | Qdrant | Milvus | Weaviate |
|----------|--------|-------------|----------|--------|--------|----------|
| AND | `And()` | All conditions true | `$and` | `must` | `and` | `And` |
| OR | `Or()` | Any condition true | `$or` | `should` | `or` | `Or` |
| NOT | `Not()` | Negate condition | - | `must_not` | `not` | - |

**Example:**

```go
filter := v.And(
    v.Eq(v.M("products", "active"), v.P("active")),
    v.Gt(v.M("products", "price"), v.P("min_price")),
)
```

## Range Filters

| Function | Description | Example |
|----------|-------------|---------|
| `Range()` | Inclusive range | `min <= value <= max` |
| `RangeExclusive()` | Exclusive range | `min < value < max` |

**Example:**

```go
minPrice := v.P("min_price")
maxPrice := v.P("max_price")
filter := vectql.Range(v.M("products", "price"), &minPrice, &maxPrice)
```
